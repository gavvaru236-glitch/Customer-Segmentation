# Customer Segmentation using Unsupervised Machine Learning

## Overview
This project segments customers into different groups based on purchasing behavior and demographic information using K-Means Clustering.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Techniques
- Data Preprocessing
- Label Encoding
- Standardization
- t-SNE Visualization
- K-Means Clustering

## Results
Identified 5 customer segments that can be used for targeted marketing and business decision-making.
